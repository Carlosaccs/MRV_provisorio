Sobre o Dashboard

Este Dashboard foi desenvolvido para facilitar a consulta rápida ao estoque e materiais dos empreendimentos MRV na Grande São Paulo e Interior.

Dicas:

  • Clique nas regiões do mapa para filtrar por cidade.
  • Utilize os botões de Copiar Link para enviar materiais rapidamente aos clientes.
  • As informações de estoque são sincronizadas com a planilha mestre.
